# 前端测试指南

## 启动服务

### 1. 确保后端运行
```bash
# 后端应该已经在运行
# 如果没有，在backend目录执行：
.\venv\Scripts\python.exe run.py
```

### 2. 启动前端服务
```bash
# 在frontend目录
npm run dev

# 应该看到
# ➜  Local:   http://localhost:5173/
```

## 常见问题

### Q1: 前端无法启动
**症状**: npm run dev 失败
**解决**: 
```bash
npm install
npm run dev
```

### Q2: 图表不显示
**症状**: 看到空白图表区域
**解决**: 检查ECharts是否正确安装
```bash
npm install echarts
```

### Q3: 样式混乱
**症状**: 页面样式不正确
**解决**: 清除浏览器缓存，硬刷新 (Ctrl+F5)

### Q4: API调用失败
**症状**: Console中显示404错误
**解决**: 确保后端服务在5000端口运行
